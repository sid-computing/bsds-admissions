# BSDS Admissions Guide

Static GitHub Pages website for `sid-computing/bsds-admissions`.

## Publish

Upload `index.html`, `styles.css`, and this README to the root of the repository. Then open **Settings → Pages**, choose **Deploy from a branch**, select `main` and `/ (root)`, and save.

## Disclaimer

This is an unofficial resource and is not affiliated with the Indian Statistical Institute (ISI).
