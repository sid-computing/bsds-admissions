# BSDS at ISI Delhi

An unofficial, student-maintained guide to the Bachelor of Statistical Data Science programme at the Delhi Centre of the Indian Statistical Institute.

The website is intended primarily for Class XI and XII students. It explains the JEE Main admission route, places historical admission data in context, and shares practical information about studying at ISI Delhi.

## Website sections

- Programme overview
- JEE Main admission route
- Eligibility and merit-list rules
- Reservation information
- Historical merit and allocation data
- ISI Delhi location, accommodation and facilities
- Academic and student life
- Preparation before joining

## Important scope note

The admission guide currently discusses the **JEE Main route**, because that is the route the author is more familiar with. ISI may also accept eligible CUET-UG scores. CUET-UG applicants should consult the latest official ISI admission notice.

Rules labelled **2027 admission cycle** are specific to that cycle and should not be assumed to apply in later years.

## Publish with GitHub Pages

1. Upload `index.html`, `styles.css` and `README.md` to the root of the repository.
2. Open the repository on GitHub.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder.
6. Click **Save**.

GitHub will display the public website address after deployment finishes.

## Editing the website

- Edit page content in `index.html`.
- Edit colours, spacing and responsive design in `styles.css`.
- Replace every “Coming soon” item only after adding verified information.
- Link factual admission claims directly to an official notice whenever possible.
- Update the “Last updated” date in the footer after making material changes.

## Design system

- Midnight blue: official and programme information
- Teal: ISI Delhi-specific information
- Amber: warnings and time-sensitive information
- Lavender: student perspective
- Silver: page background
